************************************Ficha Tecnica************************************
En esta carpeta estan los archivos clave para poder hacer la asignacion de los 
metafields a cada producto de manera masiva. 

El archivo Copia de LlantiQuick_Cat√°logo_2020 PABLO es el archivo que contiene los
productos que necesitaban informacion respecto a su ficha tecnica.

El archivo BaseDatos-VentasLinea-2021 es el archivo en el que Alma lleno la informacion
para la ficha tecnica de cada producto tomando como referencia el primer archivo 
mencionado.

El archivo Ficha_Tecnica es el archivo que se debe subir a la aplicacion maxtrify para 
poder subir la nueva informacion a la plataforma, donde se tiene asignado los nombres
de los metafields necesarios para la ficha tecnica y su integracion a la pagina.
